**Tools** is built on top of [shadcn/ui](https://ui.shadcn.com) components, providing a foundation for our library. Our components are designed to be opinionated for better consistency, while we work towards making them more flexible.

## Prerequisites

Ensure you have a project set up with shadcn/ui. If not, follow the [shadcn/ui installation guide](https://ui.shadcn.com/docs/installation) to get started.

## Installing Dependencies

For icons, install the following package:

```npm
npm install @phosphor-icons/react
```

For other components, refer to the shadcn/ui documentation to add the required base components. You can copy and paste the necessary code from their examples.

<Callout type="info">
  We're actively developing a CLI tool to integrate seamlessly with your shadcn/ui setup via registries, making installation more streamlined in the future.
</Callout>
