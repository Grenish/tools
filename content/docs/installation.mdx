## Manual Installation

If you prefer not to use the CLI or your framework is not supported, you can add components to your project manually.

### 1. Install dependencies

Components are styled using Tailwind CSS and rely on `clsx` and `tailwind-merge` for class manipulation.

```bash
npm install tailwind-merge clsx
```

### 2. Configure path aliases

We use the `@` alias to resolve imports. You need to configure this in your `tsconfig.json`.

```json title="tsconfig.json"
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./*"]
    }
  }
}
```

### 3. Add a cn helper

We use a `cn` helper to make it easier to conditionally add Tailwind CSS classes.

```ts title="lib/utils.ts"
import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

### 4. That's it

You can now start adding components to your project. Browse the documentation to find components you need, copy their source code, and paste them into your project.
