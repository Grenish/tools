Displays a button or a component that looks like a button.

<Preview>
  <Button>Button</Button>
</Preview>

## Installation

```bash
npx shadcn@latest add button
```

## Usage

```tsx
import { Button } from "@/components/ui/button"
```

```tsx
<Button variant="outline">Button</Button>
```
