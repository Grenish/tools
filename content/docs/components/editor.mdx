```tsx

  return <MainEditor />
}
```

## Features

- Rich Text Formatting: Bold, italic, underline, headings (H1, H2), bullet lists, numbered lists
- Font Selection: 10+ font families including Roboto, Montserrat, Lato, Pacifico, and more
- Undo/Redo Support: Full history management with keyboard shortcuts (Ctrl+Z / Ctrl+Y)
- Keyboard Shortcuts: Native support for Ctrl+B (bold), Ctrl+I (italic), Ctrl+U (underline)
- Link Management: Insert and manage hyperlinks with URL validation dialog
- Image Support: Embed images via URL with responsive scaling
- Live Preview Mode: Toggle between edit and read-only preview
- Intelligent Formatting: Real-time format detection and toolbar state synchronization
- History Management: Debounced history tracking with max 50 states
- Placeholder Text: Smart empty-state with contextual placeholder message
- Responsive Design: Adapts to all screen sizes with flexible toolbar
- Accessible: Semantic HTML with proper ARIA roles and keyboard navigation
- Type-Safe: Fully typed with TypeScript interfaces
