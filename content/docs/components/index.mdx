Browse the available components below. All components are installable via the `@grenish` shadcn registry. See the [installation guide](/docs/installation) for one-time setup.

## Quick Install

```npm
npx shadcn@latest add @grenish/<component-name>
```

## Anatomy

Components are built with Tailwind CSS and React. Each component ships as a self-contained file, or a small set of files when the component needs supporting pieces.

```tsx title="Example: Using registry components"
import CommentBox from "@/components/tools/comment-box"
import { GoogleButton } from "@/components/tools/google-button"
import TweetCard from "@/components/tools/tweet-card"

export default function Demo() {
  return (
    <div className="space-y-4">
      <GoogleButton />
      <CommentBox placeholder="Share your update..." />
      <TweetCard
        profileName="Grenish Rai"
        tweet="Just shipped a polished component."
        likes={2000}
      />
    </div>
  )
}
```
